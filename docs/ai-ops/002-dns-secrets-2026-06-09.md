## Passos manuais pendentes (operador)

1. **Redeploy dos apps** — os novos FQDNs e env vars so entram em vigor apos redeploy. Triggerado automaticamente pelo proximo merge na main (deploy.yml), ou manualmente via Coolify UI.
2. **Rotacionar `AUTH_SECRET`** — apos validar que o deploy esta funcionando, gerar um novo valor e atualizar o env var no Coolify.
3. **Dominio final** — quando definir o dominio definitivo (ex: `traveltogether.app`), criar novo registro DNS, atualizar FQDN no Coolify e remover o registro provisorio.
4. **SSL Full (strict)** — se quiser end-to-end TLS (Cloudflare <-> origem), habilitar no Cloudflare SSL/TLS settings e garantir que o Traefik esta com cert valido.
